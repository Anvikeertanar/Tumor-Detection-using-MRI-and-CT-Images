
# 🧠 Deep Learning Based Tumor Detection Using MRI and CT Images

## Overview

Early and accurate detection of brain tumors is critical for effective medical diagnosis and treatment. This project explores the use of Deep Learning techniques for automated tumor detection and classification using MRI and CT scan images.

A Convolutional Neural Network (CNN) model was developed to analyze medical images and classify them into different tumor categories. The system aims to assist healthcare professionals by providing a fast and reliable preliminary diagnostic tool.

---

## Objectives

* Develop a CNN-based tumor detection and classification system.
* Perform preprocessing of MRI and CT scan images.
* Train and evaluate a deep learning model for tumor classification.
* Analyze model performance using accuracy metrics and confusion matrices.
* Demonstrate the potential of AI in medical image analysis.

---

## System Workflow

```text
MRI / CT Scan Images
          ↓
 Image Preprocessing
          ↓
 CNN Model Training
          ↓
 Feature Extraction
          ↓
 Classification
          ↓
 Tumor Prediction
```

---

## Dataset

The project utilizes MRI and CT scan images containing both tumor and non-tumor cases. The dataset is preprocessed through resizing, normalization, and augmentation techniques to improve model performance and generalization.

---

## Model Architecture

A Convolutional Neural Network (CNN) architecture was implemented for feature extraction and classification.

The network consists of:

* Convolutional Layers
* Activation Functions (ReLU)
* Pooling Layers
* Fully Connected Layers
* Softmax Classification Layer

The CNN automatically learns relevant image features without requiring manual feature engineering.
